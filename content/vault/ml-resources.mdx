## some libraries

- [h3](https://h3geo.org/) — do good geospatial analysis
- [luminaire](https://github.com/zillow/luminaire) — ml driven solutions for monitoring time series data
- [mlforecast](https://github.com/Nixtla/mlforecast) — ml for time series forecasting
- [orbit](https://github.com/uber/orbit) — bayesian forecasting
- [neuralforecast](https://github.com/Nixtla/neuralforecast) — user friendly neural forecasting algorithms
- [statsforecast](https://github.com/Nixtla/statsforecast) — statistical and econometric models
- [causalml](https://github.com/uber/causalml) — causal inference with machine learning algorithms
